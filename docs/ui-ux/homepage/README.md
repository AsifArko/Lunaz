# Homepage — UI/UX Specification

## Overview

The homepage is the primary landing experience for Lunaz. It must be elegant, professional, and fully responsive across mobile, tablet, and desktop.

## Component Map

```
HomePage
├── Header (shared Layout)
├── Hero
│   ├── Tagline badge
│   ├── Main heading
│   ├── Description
│   ├── CTA buttons
│   └── Trust indicators
├── Featured Categories
│   ├── Section header
│   ├── Parent category tabs
│   └── Subcategory cards (horizontal scroll)
├── Featured Products
│   ├── Section header
│   └── Product grid
├── Promo Banner
│   ├── Tagline
│   ├── Heading
│   ├── Description
│   └── CTA
└── Footer (shared Layout)
```

## Design Principles

- Off-white gray (`#fafaf9`, `#f5f5f4`) backgrounds
- Mild transparency on overlays and glass surfaces
- Smooth 300–400ms transitions
- Futuristic minimalist aesthetic
