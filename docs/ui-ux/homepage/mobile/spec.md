# Homepage — Mobile Spec (< 640px)

## Layout

- Full-width sections, no side margins beyond container padding (16px)
- Single-column stacking
- Vertical rhythm: 48px between major sections

## Header

- Sticky, compact height (56px)
- Logo left, hamburger + cart right
- Search bar below nav row (collapsible or always visible)
- Glass-style background: `rgba(250, 250, 249, 0.9)` with backdrop-blur
- Bottom border: `rgba(231, 229, 228, 0.6)`

## Hero

- Padding: 24px vertical, 16px horizontal
- Tagline badge: full-width on very small screens, inline on 360px+
- Heading: 2.25rem (36px), line-height 1.1
- Description: 1rem, max-width 100%
- CTAs: Stacked, full-width buttons, 12px gap
- Trust indicators: 2-column grid, smaller icons (14px)

## Categories

- Section padding: 40px 16px
- Parent tabs: Horizontal scroll, snap, no scrollbar
- Tab size: min 140px width, 48px height
- Subcategory cards: 140px width, 4:3 aspect, snap scroll
- Scroll indicators: Dots below subcategories

## Products

- 1 column grid
- Card: Full width, 4:3 image
- Gap: 16px between cards

## Promo Banner

- Padding: 48px 16px
- Same structure as hero, scaled down
- CTA: Full-width

## Footer

- Stacked layout
- Brand + social at top
- Link columns: 2 columns (Shop, Account | Support, Legal)
- Bottom bar: Copyright + payment icons stacked

## Interactions

- Touch targets: min 44px
- Smooth scroll for horizontal carousels
- No hover-dependent content
