# Design Tokens — Lunaz UI/UX

Central design system values for consistent, professional, and elegant UI across the application.

## Color Palette

### Primary — Off-White Gray

| Token         | Hex       | Usage                  |
| ------------- | --------- | ---------------------- |
| `surface-50`  | `#fafaf9` | Page background, cards |
| `surface-100` | `#f5f5f4` | Elevated surfaces      |
| `surface-200` | `#e7e5e4` | Borders, dividers      |
| `surface-300` | `#d6d3d1` | Muted borders          |
| `surface-400` | `#a8a29e` | Placeholder text       |
| `surface-500` | `#78716c` | Secondary text         |
| `surface-600` | `#57534e` | Body text              |
| `surface-700` | `#44403c` | Headings               |
| `surface-800` | `#292524` | Dark headings          |
| `surface-900` | `#1c1917` | Hero, footer dark      |

### Transparency

| Token            | Value                       | Usage            |
| ---------------- | --------------------------- | ---------------- |
| `glass-bg`       | `rgba(250, 250, 249, 0.7)`  | Header, overlays |
| `glass-border`   | `rgba(231, 229, 228, 0.5)`  | Glass borders    |
| `overlay-light`  | `rgba(255, 255, 255, 0.03)` | Subtle overlays  |
| `overlay-medium` | `rgba(28, 25, 23, 0.4)`     | Dark overlays    |

## Typography

- **Display**: Cormorant Garamond, 400, -0.02em
- **Section**: Cormorant Garamond, 500, -0.01em
- **Sub**: Inter, 500, 0.05em, uppercase
- **Body**: Inter, 400, -0.01em, 1.6 line-height

## Transitions

| Token               | Value                              | Usage              |
| ------------------- | ---------------------------------- | ------------------ |
| `transition-fast`   | 150ms ease                         | Micro-interactions |
| `transition-base`   | 300ms ease                         | Hover, focus       |
| `transition-smooth` | 400ms cubic-bezier(0.4, 0, 0.2, 1) | Page transitions   |

## Spacing Scale

4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96, 128 (px)

## Border Radius

- `sm`: 6px
- `md`: 10px
- `lg`: 14px
- `xl`: 20px
- `2xl`: 24px
